Execute HTML, Text or PHP fast and easy with this Widgetify-widget.

== Description ==

Like a normal text widget this allows you to easily post text and HTML, but
now you can execute PHP too! 
This makes merging with other themes easier.

!IMPORTANT! You must use <?php and ?> tags for the code to be 
recognized.

== Installation ==

1. Upload `execphp.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use the widget like any other widget.
